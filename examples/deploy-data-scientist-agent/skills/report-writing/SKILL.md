# Report Writing

Use this skill when turning analysis into a concise report for stakeholders.

## Structure

1. **Executive summary**: 3-5 bullets with the most important findings.
2. **Method**: Input files, filters, transformations, joins, and calculations used.
3. **Findings**: Specific numbers, comparisons, and trends.
4. **Visuals**: List chart files and what each shows.
5. **Limitations**: Missing data, assumptions, small samples, or uncertainty.
6. **Recommendations**: Practical next steps grounded in the analysis.

## Style

- Keep the report direct and decision-oriented.
- Use numbers and percentages where they clarify impact.
- Separate observed facts from hypotheses.
- Do not overstate causality from correlation or aggregate trends.
- For multi-file analyses, make provenance clear: name which files supported each major finding and describe any inferred relationships.
