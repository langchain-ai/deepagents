# Python for Beginners Course

A comprehensive introductory course to learn Python programming from scratch.

## Course Structure

### Modules
- **Module 1**: Introduction to Python Basics
- **Module 2**: Control Flow and Functions  
- **Module 3**: Data Structures

### How to Use This Course

Each module contains:
- **Lesson files** in `modules/` - Read these to learn concepts
- **Exercises** in `exercises/` - Practice what you learned
- **Solutions** in `solutions/` - Check your work

### Getting Started

1. Start with Module 1 in `modules/`
2. Complete the corresponding exercises in `exercises/`
3. Compare your solutions with `solutions/`
4. Move to the next module when ready

### Prerequisites

- Python 3.8 or higher installed
- A text editor (VS Code, PyCharm, etc.)
- Basic computer literacy

### Running Code

To run any Python file:
```bash
python filename.py
```

### Time to Complete

- **Module 1**: 2-3 hours
- **Module 2**: 3-4 hours
- **Module 3**: 3-4 hours

**Total**: 8-11 hours of learning
