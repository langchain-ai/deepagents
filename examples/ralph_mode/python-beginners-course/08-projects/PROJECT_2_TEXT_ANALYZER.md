# Project 2: Text Analyzer

## Goal
Analyze a text file from the command line.

## Requirements
- Ask for a path to a text file
- Print:
  - number of lines
  - number of words
  - number of characters
  - top 10 most common words (case-insensitive)

## Hints
- Normalize by lowercasing
- Strip punctuation (start simple)
