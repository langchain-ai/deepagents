# Syllabus: Python Programming (Beginners)

Default pacing: **6 weeks**. Each week assumes ~2 sessions plus practice.

## Week 0 (onboarding)
- Setup Python, editor, terminal
- Git basics + workflow

## Week 1: Python basics
- Running Python, scripts
- Variables, types, strings, numbers
- Input/output

## Week 2: Control flow
- Booleans, comparisons
- `if`/`elif`/`else`
- Loops: `for`, `while`, `range`

## Week 3: Functions
- Defining/calling functions
- Parameters, return values
- Scope, docstrings

## Week 4: Data structures
- Lists, tuples
- Dicts, sets
- Iteration patterns

## Week 5: Files + modules + testing
- Read/write text files
- CSV/JSON intro
- Imports, packages, venv
- Debugging + `pytest`

## Projects
- Mini-project 1: CLI calculator
- Mini-project 2: Text analyzer
- Mini-project 3: To-do CLI with file persistence
- Capstone: Choose one (budget tracker / habit tracker / flashcards)

## Assessment (simple rubric)
- Correctness
- Readability
- Git hygiene (small commits, meaningful messages)
- Tests (where relevant)
