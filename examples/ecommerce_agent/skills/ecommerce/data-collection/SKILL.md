# 数据采集技能

## 何时使用
当需要采集电商平台的订单数据、销售数据、推广数据时使用此技能。

## 支持平台
- 抖音电商
- 拼多多
- 淘宝

## 采集数据类型

### 1. 订单数据
- 订单列表
- 订单详情
- 订单状态
- 收货信息
- 退款/售后信息

### 2. 销售数据
- 商品销量
- 销售额
- 销售趋势
- 热销商品排行

### 3. 推广数据
- 推广效果
- 流量来源
- 转化率
- 投入产出比

### 4. 用户数据
- 用户画像
- 用户行为
- 复购情况

## 执行步骤

### 1. 准备阶段
- 确认采集目标（日期范围
- 确定数据类型
- 准备数据存储位置

### 2. 导航到数据页面
- 进入数据中心/数据分析页面
- 选择相应的数据报表
- 设置日期筛选条件

### 3. 数据采集
- 提取页面数据
- 分页获取全部数据
- 截图保存关键数据

### 4. 数据处理
- 整理数据格式
- 数据清洗
- 数据校验

### 5. 数据存储
- 保存到数据库
- 生成数据报告

## 各平台数据页面

### 抖音电商
- **订单管理：订单列表页
- **数据中心：销售、商品分析
- **推广数据：巨量引擎

### 拼多多
- **订单管理：订单查询页
- **数据中心：经营分析
- **推广数据：多多推广

### 淘宝
- **订单管理：已卖出宝贝
- **数据中心：生意参谋
- **推广数据：直通车

## 采集频率

### 建议采集
- 订单数据：每日
- 销售数据：每日/每周
- 推广数据：每日
- 用户数据：每周

### 采集时机
- 订单数据：每日凌晨或上午
- 销售数据：每日上午统计前一天
- 推广数据：每日上午更新

## 数据格式

### 订单数据字段
```
- 订单号
- 商品信息
- 用户信息
- 订单金额
- 订单状态
- 下单时间
- 支付时间
- 发货时间
```

### 销售数据字段
```
- 日期
- 商品SKU
- 销量
- 销售额
- 访客数
- 转化率
```

## 异常处理

### 数据加载失败
- 重试1-2次
- 检查网络连接
- 检查日期范围是否正确

### 数据不完整
- 记录缺失数据
- 标记异常数据
- 补充手动补充

### 数据格式异常
- 保留原始数据
- 记录异常日志
- 尝试解析

## 安全注意事项

### 用户隐私
- 脱敏处理敏感信息
- 不泄露用户数据
- 遵守数据保护法规

### 访问频率
- 控制操作频率
- 避免触发反爬
- 保持人类行为模式
