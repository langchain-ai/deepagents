# Exercises: Basics

## Instructions
Create a branch: `git switch -c exercise/01-basics`

## A. Variables + strings
1. Create `a_strings.py`.
2. Store your first name in a variable.
3. Print: `Hello, <name>!`
4. Print the length of your name.

## B. Numbers
1. Create `b_numbers.py`.
2. Ask the user for a number (string from `input`).
3. Convert to `int`.
4. Print the number doubled.

## C. Stretch
- Ask for two numbers and print sum, difference, product.

## Git checkpoint
Commit after each section:
- "Solve strings exercises"
- "Solve numbers exercises"
