# Report Format Selection

Choose the right report structure based on the research question type:

## Comparison questions ("X vs Y", "which is better")

Use a comparison matrix:
- Criteria as rows, options as columns
- Verdict section with clear recommendation and reasoning
- Highlight where options are roughly equivalent

## Explanatory questions ("how does X work", "why does X happen")

Use a layered explanation:
- Start with the one-sentence answer
- Then the one-paragraph answer
- Then the full technical explanation
- Each layer should stand alone

## Landscape questions ("state of X", "what are the options for X")

Use a survey structure:
- Categories/segments as main sections
- Key players/approaches within each
- Trends and direction of the field
- Comparison table at the end

## Predictive questions ("what will happen", "is X a good idea")

Use an argument structure:
- Thesis statement up front
- Evidence for
- Evidence against
- Assessment of confidence
- Conditional recommendations ("if X, then Y")
