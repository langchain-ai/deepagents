# Search Strategy

When basic searches don't return good results, use these strategies:

## Query techniques

- **Quote exact phrases**: `"specific term"` to find exact matches
- **Exclude noise**: `-pinterest -reddit` to filter low-quality results
- **Site-scope**: `site:arxiv.org` or `site:github.com` for specific sources
- **Recency**: add the current year to find recent information
- **Synonyms**: try 2-3 different phrasings of the same concept

## Source prioritization

1. **Primary sources**: official docs, research papers, SEC filings, government data
2. **Expert sources**: conference talks, technical blogs by practitioners
3. **Aggregators**: Wikipedia (for overview + citations), review articles
4. **Community**: Stack Overflow, HN discussions (verify claims independently)

## When stuck

- Search for the topic in a different language or domain
- Look for "state of X 2025" or "X landscape" survey articles
- Check if there's an awesome-X list on GitHub
- Search academic databases for systematic reviews
