# Exercises: Data Structures

Create a branch: `git switch -c exercise/04-data-structures`

## A. Lists
Create `a_lists.py`:
- Start with `nums = [3, 1, 4, 1, 5]`
- Print sum, min, max
- Print a sorted copy

## B. Dicts
Create `b_phonebook.py`:
- Use a dict mapping name -> phone
- Add 3 entries
- Ask user for a name and print phone or "not found"

## C. Stretch
Create `c_word_count.py`:
- Ask user for a sentence
- Count occurrences of each word (dict)

## Git checkpoint
Commit after each file.
