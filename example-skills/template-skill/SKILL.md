# Template Skill

This is a template skill that demonstrates the basic structure of a skill for deepagents.

## Purpose

Use this template as a starting point when creating new skills. It shows the minimum required structure and common patterns.

## Structure

A skill consists of:
- **YAML frontmatter**: Contains `name` and `description` (required), plus any additional metadata
- **Markdown content**: Instructions, examples, and guidelines for the skill

## Usage Guidelines

When this skill is active, follow these guidelines:
1. Always provide clear, structured output
2. Include examples where appropriate
3. Follow best practices for the task at hand

## Examples

Example task 1: When asked to do X, respond with Y
Example task 2: When asked to do A, respond with B

## Best Practices

- Keep instructions clear and concise
- Use examples to illustrate expected behavior
- Document any assumptions or limitations
