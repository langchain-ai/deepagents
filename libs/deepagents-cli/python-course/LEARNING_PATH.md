## üéØ Learning Path

### Module 1: Introduction to Python Basics (2-3 hours)
**Topics:** Variables, data types, operations, strings, input/output

**File:** `modules/01_basics.py`

Start here if you're new to programming. You'll learn:
- What Python is and why it's popular
- How to store and use data with variables
- Different data types: int, float, str, bool, None
- Basic math and string operations
- How to interact with users via input/output

**Next Steps:**
1. Read through `modules/01_basics.py` - follow the examples
2. Run the file: `python modules/01_basics.py`
3. Complete `exercises/01_exercises.py` (10 exercises + 2 challenges)
4. Check your work with `solutions/01_solutions.py`

## üìñ How to Use This Course

### Study Method

1. **Read the Module:** Open the module file and read through all code and comments
2. **Run the Module:** Execute it to see examples in action
3. **Understand Examples:** Make sure you understand what each example does
4. **Do Exercises:** Try to solve each exercise without looking at solutions
5. **Check Solutions:** Compare your code with the provided solutions
6. **Refine Your Code:** Learn better approaches from the solutions

### Tips for Success

- **Type the code yourself** - Don't just copy/paste. Typing helps learning.
- **Run and modify examples** - Change values and see what happens
- **Take breaks** - Don't try to learn everything in one sitting
- **Debug your own code** - Use print statements to understand what's happening
- **Challenge yourself** - Try the challenge exercises and create variations
- **Ask "why?"** - Understand the reasoning behind the code, not just the syntax

### Pacing Guide

- **Fast learner:** 1 module per day = 5 days total
- **Moderate pace:** 1 module every 2 days = 10 days total  
- **Thorough learner:** 1-2 modules per week = 3-5 weeks total

Take the pace that works for you. Understanding is more important than speed.
