# Exercises: Setup + Git

## A. Terminal warmup
1. Create a folder called `practice`
2. Create a file `practice/notes.txt`
3. Add a line to it using your editor
4. Print it with `cat`

## B. Git warmup
1. Initialize a git repo in your workspace
2. Commit `notes.txt`
3. Add another line
4. Commit again

## Git checkpoint
- Branch name: `exercise/00-setup-git`
- Commit messages:
  - "Add notes file"
  - "Update notes"
