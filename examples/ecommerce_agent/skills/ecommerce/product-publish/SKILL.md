# 商品发布技能

## 何时使用
当用户请求发布商品到抖音电商、拼多多、淘宝等电商平台时使用此技能。

## 支持平台
- 抖音电商
- 拼多多
- 淘宝

## 执行步骤

### 1. 前置准备
- 检查是否已登录目标平台
- 如果未登录，先进行登录操作
- 确认当前店铺信息

### 2. 导航到商品发布页
- 根据平台导航到相应的商品发布页面
- 抖音："商品管理" -> "发布商品"
- 拼多多："商品列表" -> "发布商品"
- 淘宝："出售中的宝贝" -> "发布宝贝"

### 3. 选择商品类目
- 选择正确的商品类目
- 参考历史经验选择适合的类目

### 4. 填写商品信息
- 商品标题：包含关键词，符合平台规范
- 商品描述：详细描述商品特点和规格
- 商品价格：设置合理的销售价格
- 库存数量：设置库存
- 商品规格：配置颜色、尺寸等

### 5. 上传商品图片
- 上传商品主图（至少1张）
- 上传商品详情图
- 注意图片尺寸和格式要求

### 6. 提交审核
- 检查商品信息完整性
- 提交商品审核
- 等待审核结果

## 操作规范

### 人类行为模式
- 填写表单时使用打字延迟，模拟真实用户
- 点击间隔合理，避免操作过快
- 遇到验证码时暂停，通知用户处理

### 平台规则
- 遵守各平台的商品发布规范
- 避免违规关键词
- 确保商品信息真实准确

### 安全检查
- 发布前检查商品标题
- 检查图片内容合规性
- 确认价格设置合理

## 异常处理

### 常见问题
1. **登录失败**
   - 暂停操作，通知用户
   - 记录失败状态

2. **类目选择错误**
   - 重新选择类目
   - 参考历史经验

3. **图片上传失败**
   - 重试1-2次
   - 如果仍失败，使用备用图片

4. **提交失败**
   - 查看错误信息
   - 根据提示修改后重试

### 断点恢复
- 支持从任何步骤恢复
- 自动保存进度到任务中

## 示例流程

```
用户：帮我把"新款连衣裙"发布到抖音
1. 检查登录状态
2. 导航到抖音商品发布页
3. 选择"女装/连衣裙"类目
4. 填写商品信息
5. 上传连衣裙图片
6. 提交审核
```
