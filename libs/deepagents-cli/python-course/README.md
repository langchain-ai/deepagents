### Module 1: Introduction to Python Basics (2-3 hours)

- Methods and attributes

- Practical OOP design
- **450 lines of lesson code + 10 exercises**

### Final Project: Todo Application (1-2 hours)
- Complete command-line todo app with file persistence
- Demonstrates all 5 modules combined
- Add, remove, complete, search, save, load tasks
- ~250 lines of production-quality code
