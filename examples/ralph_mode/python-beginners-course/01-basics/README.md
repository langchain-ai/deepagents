# Module 1: Python Basics

## Objectives
- Run Python files
- Understand variables and basic types
- Use `print()` and `input()`

## Topics
- Running scripts: `python file.py`
- Variables: `name = "Ada"`
- Types: `int`, `float`, `str`, `bool`
- Converting types: `int("123")`

## Demo
Create `hello_name.py`:
```py
name = input("What is your name? ")
print("Hello,", name)
```
