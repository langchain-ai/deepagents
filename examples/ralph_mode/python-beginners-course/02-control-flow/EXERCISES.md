# Exercises: Control Flow

Create a branch: `git switch -c exercise/02-control-flow`

## A. Conditionals
Create `a_grade.py`:
- Ask for an integer score 0	100
- Print A/B/C/D/F based on ranges

## B. Loops
Create `b_countdown.py`:
- Ask for a starting integer
- Count down to 0

## C. Stretch
Create `c_guess.py` (guessing game):
- Secret number is 7
- Keep asking until correct
- Tell the user if too high/low

## Git checkpoint
Commit messages:
- "Add grade classifier"
- "Add countdown"
- "Add guessing game"
