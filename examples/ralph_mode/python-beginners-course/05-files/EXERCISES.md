# Exercises: Files

Create a branch: `git switch -c exercise/05-files`

## A. Read a file
Create `a_read.py`:
- Read `sample.txt` and print number of lines

Create `sample.txt` with at least 5 lines.

## B. Write a file
Create `b_write.py`:
- Ask user for 3 lines of text
- Save them to `output.txt`

## C. Stretch (JSON)
Create `c_json.py`:
- Save a dict to `data.json`
- Load it back and print a value

## Git checkpoint
Commit after each section.
