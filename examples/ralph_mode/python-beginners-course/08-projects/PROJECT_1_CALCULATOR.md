# Project 1: CLI Calculator

## Goal
A terminal calculator you can use repeatedly.

## Requirements
- Ask for number A
- Ask for operator (+ - * /)
- Ask for number B
- Print result
- Keep running until user types `q`

## Edge cases
- Division by zero
- Non-numeric input
